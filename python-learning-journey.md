# My Python Learning Journey

I am trying to build my confidence step by step.

At first I wrote very simple programs. Then I started learning how functions make code easier to understand and test.

Later I explored OOP and noticed that better structure helps both readability and testing.

One important thing I noticed is that better structure usually makes testing easier.

I also started thinking more from a tester's point of view by asking:
- What can break?
- What are the edge cases?
- How can I verify behaviour with tests?

My next target is to keep building small projects and add tests where possible.
